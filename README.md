# hex-gram-schmidt

Integer Gram-Schmidt orthogonalization for the `hex` project. Mathlib-free.

```
require HexGramSchmidt from git "https://github.com/kim-em/hex-gram-schmidt.git" @ "<rev>"
```

- Depends on [`hex-matrix`](https://github.com/kim-em/hex-matrix).
- Mathlib correspondence: [`hex-gram-schmidt-mathlib`](https://github.com/kim-em/hex-gram-schmidt-mathlib).
- Benchmarks in [`bench/`](bench); conformance in [`conformance/`](conformance) (own lakefiles).
- Spec: [SPEC/hex-gram-schmidt.md](SPEC/hex-gram-schmidt.md).

Development happens in [`hex-dev`](https://github.com/kim-em/hex-dev).
